## 3 分钟跑起来

### 方式一：EXE 单文件（推荐）

双击 `金榜Agent.exe`，黑窗口出现后双击 `打开我.html`。

> 无需安装 Python，自带运行环境。

### 方式二：解压即用

解压 `金榜Agent-便携版.zip` → 双击 `启动.bat` → 双击 `打开我.html`。

> 同样无需安装任何东西。

### 方式三：源码运行

```
1. 装 Python（python.org，勾 Add to PATH）
2. 下载本项目 ZIP → 解压
3. 双击 启动.bat
4. 双击 打开我.html → 右上角 API 设置 → 填 DeepSeek Key
5. 在查询栏输入省份、分数、位次、专业 → 开始智能分析
```

## 界面

三栏工作台设计，不是聊天窗口：

- **顶部查询栏** — 省份、分数、位次、专业偏好，填好点「开始智能分析」
- **三栏结果区** — 冲一冲 / 稳一稳 / 保一保，每所学校一张卡片，分数位次年份一目了然
- **底部追问面板** — 对结果有疑问？展开后直接问 AI 顾问
- **历史记录抽屉** — 左上角「历史记录」，所有查询自动保存，随时回看
- **🌓 明暗切换** — 顶部栏一键切
- **API 设置** — Key 存浏览器本地，不上传

## 核心能力

| 功能 | 说明 |
|------|------|
| 🗄️ **本地数据库** | 14 省 × 2024-2025 年官方投档线，约 24.8 万条 |

```
你填写: 省份=浙江  位次=5000  专业=计算机

         ↓
```

## 常见问题

**数据准不准？** — 来自各省考试院官方投档线，多轮联网交叉验证。录取数据每年变，最终以官方为准。

**怎么更新？** — 重新下载 ZIP/EXE 覆盖。

**要钱吗？** — 金榜 Agent 免费开源。AI 调用按量计费，DeepSeek 聊几百次才几毛钱。

**能换别的 AI 吗？** — 能，API 设置里改 Base URL 和 Model。

**EXE 被 Windows 拦截？** — 右键 EXE → 属性 → 底部「解除锁定」。
